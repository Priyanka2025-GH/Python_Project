# Python Mini Projects

This repository contains beginner-friendly Python projects developed to practice core Python programming concepts such as functions, loops, dictionaries, conditional statements, and user input handling.

## Projects Included

### 1. Number Search
A menu-driven program that allows users to search numbers in a list using different search operations.

Features:
- Exact number search
- Search numbers greater than a given value
- Search numbers less than a given value
- Search numbers within a specified range

Concepts Used:
- Lists
- Loops
- Functions
- Conditional Statements

---

### 2. Student Marks Analyzer
A Python application that stores student marks and performs basic analysis.

Features:
- Calculate average marks of each student
- Find the topper
- Display subject-wise average marks

Concepts Used:
- Dictionaries
- Loops
- Mathematical Operations
- Conditional Statements

---

### 3. Contact Book
A simple command-line Contact Book application.

Features:
- Add new contacts
- View all contacts
- Search contacts
- Update contact details
- Delete contacts

Concepts Used:
- Dictionaries
- Functions
- Loops
- String Manipulation

---

### 4. Banking System
A basic banking system that simulates common banking operations.

Features:
- Create an account
- Check account balance
- Deposit money
- Withdraw money
- Menu-driven interface

Concepts Used:
- Dictionaries
- Functions
- Loops
- Conditional Statements
- Arithmetic Operations

---

## Technologies Used

- Python 3

These projects were created to strengthen the understanding of:
